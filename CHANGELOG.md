## [Unreleased]

### Added
- Текстовый обработчик отправляет промежуточное сообщение «📝 Разбираю…» во время парсинга, по аналогии с фото/голосом.
- Все статусные сообщения (текст / фото / голос) привязаны reply'ем к исходному сообщению пользователя — в групповом чате видно, на какую реплику отвечает бот.
- Поддержка direct-link Mini App для групповых чатов: новая env-переменная `WEBAPP_SHORT_NAME`, в группе под ответами появляется URL-кнопка вида `https://t.me/<bot>/<short_name>`. В приватных чатах поведение прежнее (`web_app=` инлайн-кнопка).
- Автоматический приветственный пост с кнопкой Mini App при добавлении бота в групповой чат (`my_chat_member`); сообщение пинится, если у бота есть `can_pin_messages`.

### Fixed
- В групповом чате финальное подтверждение «✓ Добавил N товаров» теперь действительно появляется в чате. До этого `sendMessage`/`editMessageText` падал с ошибкой Bot API: inline-кнопка `web_app=` запрещена вне приватных чатов, исключение всплывало после записи в БД, и в итоге товары добавлялись, а сообщения от бота пользователь не видел. Клавиатура теперь подбирается по типу чата.

## [0.1.0] – 2026-04-29

### Added
- Bootstrap бота «Список покупок»: text/voice/photo ингест через OpenAI (Whisper + gpt-4o), aiogram-бот, FastAPI mini-app, SQLite-хранилище, whitelist-доступ, общий активный список с авто-архивированием при отметке всех товаров.
